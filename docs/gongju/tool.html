# nvm

nvm为nodejs版本管理工具,来帮助我们更好的使用nodejs,它可以帮助我们切换不同的nodejs版本

ps.安装之前请注意,如果电脑上已经安装了nodejs,需要先进行卸载

nvm下载网址:Releases · coreybutler/nvm-windows (github.com)open in new window

常用命令:

查看所有已安装的nodejs版本

nvm list

查看nodejs 官方的所有版本

nvm list available

分别为:当前的 |长期支持的|旧的稳定的|旧的不稳定的

下载对应的node版本号

nvm install 版本号

切换node版本

nvm use 版本号

# nrm

nrm为npm源管理器,,它可以帮助我们切换不同的npm源

1.全局安装

npm i -g nrm

2.查看版本

 nrm -V

3.查看所有源

nrm ls 			//查看所有源
  npm ---------- https://registry.npmjs.org/
  yarn --------- https://registry.yarnpkg.com/
  tencent ------ https://mirrors.cloud.tencent.com/npm/
  cnpm --------- https://r.cnpmjs.org/
  taobao ------- https://registry.npmmirror.com/
  npmMirror ---- https://skimdb.npmjs.com/registry/
nrm current 	//查看当前源

4.切换源

nrm use <registry>	//registry即是上面命令所列出来的名称

5.添加源

nrm add <registry> <url>

6.删除源

nrm del <registry>

7.测试源的响应时间

nrm test <registry>
