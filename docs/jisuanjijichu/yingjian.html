硬件简介

汪锦城2023/11/6

# 主板

是电脑各个配件的连接平台。主板的大小直接关系到机箱的物理大小和硬件的可组装大小。

型号 例如：华硕 TUF GAMING Z790-PLUS D5 WIFI

我们需要关注的是Z790

其中首字母表示：

首字母ZBH
Intel高端中高端中低端
首字母XBA
AMD高端中高端中低端

其后的第一位数字Z790表示系列数：

  • Intel已经出到7系列（显然这块主板为Intel）
  • AMD已经出到6系列

最后两位数字Z790绝大多数情况下表示性能好坏

  • 同系列数字越大越好

主板高低端的差距，主要体现在：

  • PCI-E的数量和规格
  • I/O接口的数量和规格
  • 内存D4/D5
  • 内存/CPU超频
  • 供电

说人话就是：

  • 能装什么
  • 有多能装
  • 能不能hold住
  • 能不能榨干

# CPU

也叫中央处理器（CPU，central processing unit），是电子计算机的主要设备之一 ，作为计算机系统的运算和控制核心，是信息处理、程序运行的最终执行单元。

即是计算机的大脑

主要厂商：

  • Intel
    • 以i3、i5、i7、i9开头的，为Intel酷睿
    • 性能上i3<i5<i7<i9,但只适用于同代
    • 7代i7-7700k与8代i5-8400性能不相上下
  • AMD
    • 以R3、R5、R7、R9开头的，为AMD锐龙
    • 性能上R3<R5<R7<R9,但只适用于同代

命名规则基本如同主板，例如i7-7700K：

  • 第一位代表几代
  • 后三位代表性能，同系列越大越好

后缀：

  • AMD
    • 无后缀：无核显（7000系除外）
    • X: 高频版
    • X3D：高L3缓存版（打游戏好）
    • G：带Vega核显版
  • Intel
    • 无后缀：有核显 不超频
    • K：可超频（不锁倍率）
    • F：无核显
    • S：特别版
    • T：超低功耗
    • X/XE：强无敌

性能加强技术：

提高核心

即有多少人干活，1人为单核，2人为双核...以此类推

超线程技术

本质是提高核心的资源使用效率

即以前1个核心对应1个线程，现在使一个核心不止对应一个线程，从而提高CPU性能

频率

主频相当于平时使用的速度

睿频相当于灵活调整速度，重要的事情做快点

超频要求以极限速度工作

# 显卡

​ 是计算机中一个很重要的组成部分 ，用途是将计算机系统所需要的显示信息进行转换驱动显示器，并向显示器提供逐行或隔行扫描信号，控制显示器的正确显示。

​ 这个是专业解释，简单来说，显卡就是把存储的数字信息变成图片或者视频等数据，交给显示器显示出来。因为负责显示，所以对于喜欢玩游戏和从事专业图形设计的人来说，显卡非常重要。

分类：

  • 核显
    • 集成于CPU
    • office，小型游戏。性能足够
  • 独显
    • 独立硬件
    • 建模，设计，3a游戏。

生产厂家：

  • 英伟达（市场占有更高，80%左右）
    • 以GTX或RTX开头，几代目+数字大越好+后缀
    • 从20代开始，全部为RTX，意为支持光线追踪（光线在水面的反射更加清晰）和DLSS技术（AI将低清渲染成高清画面,节省算力来提高帧数）
    • TI后缀意为在中间，如3060TI性能在3060和3070之间
  • AMD
    • 以RX开头，几代目+数字大越好+后缀
    • 优化主要在软件层面

独显 Hdmi/DP/USB-C线，务必插在显卡上（别插主板上）

# 内存

​ 因为硬盘太大，如果当CPU在工作时，每次都需要从硬盘等外部存储器上读取数据，会大大的影响工作效率！为了解决这个问题，人们便在CPU与外部存储器之间，建了一个“临时小仓库”----------内存。

​ 内存虽然容量不大，一般只有几百MB到几GB，但速度非常快，如此一来，当CPU需要数据时，事先可以将部分数据存放在内存中，以解CPU的燃眉之急。由于内存只是一个“中转仓库”，因此它并不能用来长时间存储数据，关机后数据即消失。

​ 这个很好理解，如果没有内存的存在，我们在编辑一个文档的时候，每输入一个字，那么都要往硬盘保存一次，速度非常慢。现在有了内存，我们可以先编辑，那么这些内容暂时放在内存中，但是不能永久保存，等完成一部分内容时，再保存到硬盘中。

容量（即内存多大）：

  • office，短视频：8G
  • 游戏玩家：16G（2个8G内存条）
  • 专业需求：32G（4个8G内存条）

频率（即速度多块）：

  • 主流DDR4，新一代DDR5
  • 数值越大越快
  • 玩游戏并不一定是越快越好

时序（延迟多少）：

  • 频率越大越高，但同频下越少越好

# 硬盘

硬盘是一种最为常见的外存储器，简单理解就是用来存储数据的。计算机中所有的文件，图片，音乐，电源等都是存储在硬盘中的。存储在硬盘中的数据可以永久化存储。

​ 当我们编辑一个文件，点击保存按钮的时候，实际上计算机做的就是把这个文件保存到硬盘上。也叫做将数据写到硬盘。

​ 当我们打开一个文件时，实际上计算机做的就是把这个文件的数据从硬盘中获取出来。也叫做从硬盘读取数据。

机械（磁盘）

  • 依靠机械结构，转动查找
  • 很忙很慢

固态

  • 电子交互，速度超快，机械速度十倍不止
  • 1TB SSD固态硬盘 M.2接口（NVMe协议PCIe 4.0 X4）
    • 1TB为容量：512 1TB皆可，视自身情况而定
    • 协议：PCIe接口的一般为NVM协议
    • 接口：
      • SATA走传统通道，理论最高600mb/s,限速500多mb/s
      • M.2可走传统通道，但走PCI-e通道（高速）
        • 3.0 4GB/S
        • 4.0 8GB/S

# 电源

​ 电脑电源是安装在机箱内的封闭式独立部件，它不是直接将电传输给CPU，显卡，内存，硬盘等各个配件。而是将交流电变换为稳定可靠的直流电，供给主机箱内的主板。再由主板给CPU，显卡，内存，硬盘等各个配件供电。

  • 瓦数尽量选高一点
  • 金牌，银牌电源，比的是交流电转换为直流电的转换效率

# 显示器

每一个像素点都是一个小方块。显示器所能显示的像素点越多，画面就越精细。

分辨率：

  • 分辨率1920*1080为例，表示在显示器上，长为1920个像素点，宽为1080个像素点。

  • 1920x1080 ，1998x1080 ，2048x1080 ，2048x858 这四种分辨率都称之为2K屏。

  • 3840x2160，3996x2160 ，4096x2160， 4096x1716 这四种分辨率都称之为4K屏

  • 分辨率是个非常重要的性能指标之一。

# 键盘鼠标

更加方便的使用计算机

  • 键盘

    • 薄膜键盘：便宜耐用
    • 机械键盘：打游戏手感好，按键时劈里啪啦

  • 鼠标

    • 选择适合自己的即可

# 机箱

不管是笔记本电脑还是台式机电脑，机箱其实就是一个金属外壳，把各种配件包在一起的。

# 光驱

电脑用来读写光碟内容的机器。光驱可分为CD-ROM驱动器、DVD光驱(DVD-ROM)、康宝(COMBO)、蓝光光驱（BD-ROM）和刻录机等。

​ 针对于大多数电脑而言，光盘仅仅可以安装系统，观看电影等。但是这些功能渐渐已经被U盘所替代了。所以大多数电脑上基本已经不需要用到光驱了。

仍有些特殊用途需要光驱，尚未完全淘汰
